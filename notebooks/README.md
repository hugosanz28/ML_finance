# Notebooks

Este directorio queda para trabajo exploratorio y material histórico.

- `old/`: notebooks anteriores conservados como referencia.
- raíz de `notebooks/`: espacio para nuevas exploraciones puntuales, siempre que no sustituyan al código reusable en `src/`.

Regla práctica:

Si una lógica deja de ser solo exploratoria y pasa a ser parte del producto, debe migrarse a módulos Python en `src/`.
