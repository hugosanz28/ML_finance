# monitor_tematico

Objetivo:

Seguir noticias y eventos sobre activos, sectores y geografías relevantes para la cartera.

Entradas previstas:

- posiciones actuales,
- watchlist,
- sectores y geografías expuestas,
- informe base reciente.

Salidas previstas:

- resumen de eventos relevantes,
- riesgos o catalizadores detectados,
- enlaces y fecha de las fuentes consultadas.
