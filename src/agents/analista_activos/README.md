# analista_activos

Objetivo:

Analizar las inversiones existentes y candidatas.

Cobertura prevista:

- para acciones: negocio, métricas, riesgos y cambios relevantes,
- para ETFs: proveedor, índice, holdings principales, sectores y sesgos geográficos.

Salidas previstas:

- ficha resumida por activo,
- cambios respecto a revisiones anteriores,
- observaciones útiles para seguimiento periódico.
